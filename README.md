## Lab 3

### App Description
Flashbomb saves user input (Question and Answer) and displays it on a flashcard. Users can create multiple flashcards and the data is saved when the app is restarted. Flashcards can also be deleted. 

### App Walk-though

<img src="https://imgur.com/1iNz3Qc.gif" width=200><br>


## Required
- [x] User can create a card and still see their created card when the app is relaunched.
- [x] User can create muliple cards and browse through all created cards

## Optional
- [x] User can delete a card and no longer see it in their deck anymore
- [ ] Flashcards are shown in random order instead of the order they were created in
- [ ] User can edit a card and see the edit saved when they browse through their deck of cards
- [ ] User can create flashcards with multiple answers and be able to browse through cards with multiple choice answers shown
